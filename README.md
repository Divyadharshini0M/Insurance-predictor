# 🏥 Insurance Cost Predictor (ML Project)

This project predicts **medical insurance cost** using Machine Learning.
The model takes user details such as age, BMI, smoking status, region, etc., and predicts the **insurance amount in Indian Rupees (₹)**.

---

## 🚀 Features

* Predict insurance cost using ML model
* Clean and responsive frontend UI
* Dark theme interface
* INR (₹) currency output
* Flask-based web application
* Simple and easy to run

---

## 🧠 Machine Learning

The model is trained using:

* Linear Regression / Random Forest (depending on your model)
* Scikit-learn
* Pandas & NumPy

Input Features:

* Age
* Gender
* BMI
* Children
* Smoking status
* Region

Output:

* Predicted Insurance Cost (₹)

## ⚙️ Installation

Install required packages:

```
pip install flask numpy pandas scikit-learn joblib
```

---

## ▶️ Run the Project

```
python app.py
```

Server will start at:

```
http://127.0.0.1:5000
```

Open in browser.

---

## 💻 Usage

1. Enter user details
2. Click **Predict Insurance**
3. View predicted insurance cost in **₹ Rupees**
